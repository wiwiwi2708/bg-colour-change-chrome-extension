# bg-colour-change-chrome-extension
This repository contains my first Chrome Extension built to understand how Chrome extensions work. It demonstrates the basics of extension architecture, the Chrome Extensions API, popup and content scripts, permissions, and script injection through a simple background color changer for the active webpage.

## What I Learned
- Chrome Extension architecture
- Purpose of `manifest.json`
- Popup UI (`popup.html` + `popup.js`)
- Content scripts and DOM manipulation
- Chrome Extension APIs (`chrome.tabs`, `chrome.scripting`)
- Extension permissions (`activeTab`, `scripting`)
- Loading and testing unpacked extensions in Chrome
- Debugging Chrome extensions using DevTools
